    Dynamic Shopping Cart

A lightweight, vanilla JavaScript web application that simulates a dynamic shopping cart. Users can add products, adjust quantities, and see the total price update in real-time.

Features:
Add Products: Input a custom product name and price to add it to the cart.
Input Validation: Prevents adding items with empty names, negative prices, or zero values.
Dynamic Quantity Management: Adjust the quantity of items directly in the cart, with the total price recalculating automatically.
Remove Items: Delete individual items from the cart and instantly update the total cost.
